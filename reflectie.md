Wat is een LLM?
Een LLM is een Large Language Model, dit is een vorm van kunstmatige intelligentie die is gemaakt om met taal te werken. Het model is getraind met enorme hoeveelheden tekst, zoals websites, boeken en artikelen. Hierdoor leert het patronen in taal herkennen. Een LLM kan daardoor vragen beantwoorden, teksten schrijven, samenvattingen maken en helpen bij programmeren. Het lijkt soms alsof het model echt begrijpt wat je zegt, maar dat is niet zo. In werkelijkheid voorspelt het steeds welk woord het meest logisch als volgende komt. Omdat het zo veel voorbeelden heeft gezien, kan het vaak goede en duidelijke antwoorden geven. Toch blijft het een computerprogramma en geen mens.

Risico voor developers
Een groot risico voor developers is dat ze te veel vertrouwen op de output van een LLM. Het model kan namelijk fouten maken of informatie verzinnen die heel betrouwbaar klinkt. Als een developer deze informatie direct gebruikt in code, kan dit zorgen voor bugs, verkeerde logica of zelfs beveiligingslekken. Ook kan een LLM verouderde informatie geven. Daarom is het belangrijk dat developers altijd zelf blijven nadenken, testen en controleren wat een LLM oplevert. Zo voorkom je problemen en maak je betere en veiligere software.
